Project Description:
This project develops an AI-driven web system that automatically detects fraudulent academic documents using OCR and deep learning. Users can upload certificates, and the system extracts text, analyzes visual patterns, and identifies signs of tampering such as altered text, fake signatures, or layout inconsistencies. A CNN-based model classifies documents as genuine or fake with a fraud probability score. The system aims to provide fast, accurate, and reliable document verification for universities, HR teams, and organizations.
